# Registration Form App

This project is a simple registration form built using vanilla JavaScript, HTML, and CSS. It includes fields for user name, email, password, and password confirmation, along with client-side validation to ensure all fields are filled out correctly.

## Project Structure

```
registration-form-app
├── src
│   ├── index.html          # The main HTML file containing the registration form
│   ├── styles
│   │   └── style.css       # CSS file for styling the registration form
│   └── scripts
│       └── form-validation.js # JavaScript file for form validation
├── README.md               # Documentation for the project
└── package.json            # Configuration file for npm
```

## Features

- User-friendly registration form
- Input validation for:
  - Required fields
  - Valid email format
  - Password confirmation
- Responsive design for various devices

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd registration-form-app
   ```

2. **Open the project**:
   Open `src/index.html` in your web browser to view the registration form.

3. **No additional dependencies**:
   This project does not require any additional npm packages. However, you can run `npm install` if you want to set up a package manager for future enhancements.

## Usage

Fill in the registration form with the required information. The form will validate the inputs and provide feedback if any errors are found. Ensure that:

- All fields are filled
- The email is in the correct format
- The password matches the confirmation
